Da es nur sehr wenig Spiele für Enigma2 gibt, habe ich mein E1 Sudoku Plugin auf Python umgesetzt. Die Klasse zur Generierung eines neuen Sudoku stammt von Robert Wohleb.

Die Bedienung ist sehr einfach:
Mit der "<" und ">" Taste kann die Spielstärke umgeschaltet werden, durch Drücken der "0" wird das aktuelle Feld gelöscht.

Bei Verlassen des Spiels wird der Spielstand im Pluginverzeichnis abgespeichert und beim nächsten Start automatisch wieder geladen...

...have Fun!
